# ARFITEC ToF Analysis Software

**Version:** 1.0

---

## Description

ARFITEC ToF Analysis Software is a graphical application developed for the analysis of neutron Time-of-Flight (ToF) experiments and Neutron Activation Analysis (NAA).

The software provides tools to:

- Load and visualize multiple experimental datasets.
- Apply detector efficiency and dead-time corrections.
- Display neutron flux in both Time-of-Flight and Energy domains.
- Perform Maxwellian and Maxwellian + Epithermal fits.
- Compute neutron cross sections.
- Export publication-quality figures.

---

## Main Features

- Multiple dataset visualization
- Corrected ToF flux
- Energy spectrum conversion
- Least-square Maxwellian fitting
- Advanced nonlinear fitting
- Cross-section analysis
- Interactive plotting tools
- Figure history
- Data export
